# Web-Based Inventory Management System

## Description
This project is a responsive web-based inventory and online shopping management system developed as part of a four-member academic team. The system allows management of inventory items and simulates an online shopping interface.

## Features
- Add and manage inventory items  
- Responsive user interface  
- Basic product display and shopping workflow  
- Organized layout for easy navigation  

## Technologies Used
- HTML  
- CSS  
- JavaScript  

## My Contribution
- Designed and implemented front-end components  
- Assisted in functionality implementation  
- Contributed to UI structuring and testing  

## How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in a web browser.
